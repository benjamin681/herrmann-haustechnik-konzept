# Herrmann Haustechnik GmbH — Website-Konzept

Premium Website-Konzept für die Herrmann Haustechnik GmbH, Rosenberg.

**Erstellt von:** [LaneCore AI](https://lanecoreai.com)

## Vorschau

Das Konzept zeigt eine moderne, hochwertige Webpräsenz mit folgenden Seiten:

- **Startseite** — Cinematic Hero, Teamfoto, Leistungsübersicht, Fuhrpark
- **Leistungen** — Interaktives Akkordeon mit allen 6 Kernbereichen + Fördermittel-Highlight
- **Über Uns** — Geschäftsführer-Profile, Teamfoto, Unternehmenswerte
- **Karriere** — Stellenangebote mit WhatsApp-Direktbewerbung (30 Sekunden)
- **Kontakt** — Kontaktdaten, Formular, WhatsApp-Schnellkontakt

## Technologie

- React (JSX) — Single-File-Komponente
- Scroll-Animationen via IntersectionObserver
- Responsive Design
- WCAG AA Accessibility
